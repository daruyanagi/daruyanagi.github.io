# CLAUDE.md — daruyanagi.github.io

Claude がこのリポジトリで作業する際のルール。

## 校正

誤字・脱字チェックに加えて、日本語文中の**英単語・数字の前後に半角スペースがあるか**を確認する。

- 例: 「Claude」→「 Claude 」、「v1.1.0」→「 v1.1.0 」
- ただし文頭・文末・括弧の直内側など、スペース不要な位置は除く

## 新規記事のフロントマター

新しいブログ記事を作成するときは、フロントマターに `images:` セクションを必ず含める。

```yaml
---
date: ...
title: ...
tags: []
images:
- image1.png
draft: true
---
```

`images:` は SNS（Twitter/X など）で記事が引用されたときのアイキャッチ画像として必要。抜けていると SNS カードに画像が表示されない。画像がまだ決まっていなければプレースホルダーを入れて後で差し替える。

## 掲載前チェック

「掲載前チェック」（または「公開前チェック」「掲載チェック」など）と指示されたら、以下をまとめて自動実行する。

1. **校正**: 誤字・脱字チェック＋英単語・数字の前後の半角スペース確認
2. **下書き解除**: フロントマターの `draft: true` を `draft: false` に変更
3. **タグ提案・追加**: 記事内容に合ったタグを `tags:` に追加。タグは**英数字**で記述する。既存タグは https://daruyanagi.jp/tags/ を参照し、必要なら新規タグを提案する
4. **ALT テキスト確認**: 画像の ALT テキストが不適切（空・`alt text` のままなど）な場合は修正案を提案する
5. **アイキャッチ画像確認**: フロントマターに `images:` セクションがあるか、記載ファイルが記事ディレクトリに実際に存在するかチェックし、不足・不一致があれば報告する

## 追記

「追記して」と依頼されたら、記事の末尾に以下の見出しを追加してから内容を書く。

```
## 追記（YYYY-MM-DD HH:MM）
```

日付は依頼された当日、時刻は `date` コマンドで取得した実時刻を 30 分単位に丸めて使う（例: 15:28 → 15:30）。
